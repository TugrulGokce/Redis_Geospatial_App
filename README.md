# Installation

## Docker Redis Setup

#### Pulling Redis Official Image from Docker Hub
**docker pull redis**

#### Converting Docker Image to Container and Start Redis
**docker run --rm -i -t -p 6379:6379 --name local-redis redis**


## Python Libraries
**pip install -r requirements.txt**

# Starting Streamlit

**streamlit run .\main.py**
